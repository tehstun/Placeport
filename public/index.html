PlacePort

This is PlacePort. Placeport will generate a 'placeholder' image upon request, given a width and a height.

The width and height are specified in the image path:

/img/{width}/{height}

For example, a 400x200 image would be specified as:

/img/400/200

By default, the image has colour blocks that are 50x50 pixel squares. We can control how large the blocks are with the size parameter:

/img/400/200?square=20

The server has limits on dimensions and square size, for performance reasons. Request outside the allowed dimensions will fail; requests with square size under the minimum will use the minimum square size.

The centre of the image shows the image dimensions. This can be replaced with any text:

/img/400/200?text=PlacePort

More examples, combining custom text and square sizes:

/img/240/180?square=50&text=Fifty

/img/240/180?square=20&text=Twenty

/img/240/180?square=1&text=One
